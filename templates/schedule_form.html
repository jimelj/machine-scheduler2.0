Scheduling Algorithm

The scheduling algorithm works by:

  • Analyzing the inserts required for each zip code
  • Calculating the overlap between consecutive zip codes
  • Assigning zip codes to machines to maximize insert continuity
  • Generating pocket change instructions for operators

This approach minimizes the need to change inserts between jobs, improving operational efficiency.

Note:

Make sure you have uploaded the Insert Orders file before generating a schedule. The system will use the Zips by Address File Group to determine if a ZIP is scheduled for Monday or Tuesday.
